Cost

KubeCost cluster metrics

Dashboard integrated with KubeCost providing a view on findings and insights. This dashboard gives you Kubernetes cluster costs:

  • Cluster Wide (Live and Estimative)
    • Relative price of spot instances
  • Namespace (Live and Estimative)
    • Price variation between days and weeks
  • APP (Live and average)
    • Price comparison with 7 days ago
  • PVC Costs

Node.js

Node.js metrics

Based on this template

This dashboard works with the metrics exported by the prom-client package for node.js. CG DevX Log management is pre-configured to automatically collect metrics exported by node.js application.
